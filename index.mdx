ZilMate is a programmable multi-agent AI runtime you install with a single `npm install zilmate`. It gives your applications a Manager agent, 30+ specialist swarm agents, long-term memory, background job scheduling, voice control, and a full suite of host utility tools — all accessible via a TypeScript SDK or an interactive CLI.

## What you can build

<Steps>
  <Step title="Install and initialize">
    Add ZilMate to any Node.js project with `npm install zilmate` and call `createZilMate()` to get a fully configured agent session.
  </Step>
  <Step title="Connect your first agent">
    Send a task to the Manager agent. It automatically routes to the right specialist — coding, research, image generation, finance, and more.
  </Step>
  <Step title="Add memory and persistence">
    Use `remember()` and `recall()` to give your agents long-term context across sessions, or publish knowledge to the shared Corporate Wiki for swarm-wide access.
  </Step>
  <Step title="Schedule background work">
    Create cron-based jobs that run automatically, with QStash webhooks for reliable delivery in hosted environments.
  </Step>
</Steps>

<Note>
  ZilMate requires Node.js 20 or higher and a valid `AI_GATEWAY_API_KEY`. Run `zilmate setup` after installation for an interactive configuration wizard.
</Note>
